# Campus Placement Prediction Project

## Overview
This project analyzes campus placement data and builds a machine learning model to predict student placement outcomes.

## Project Structure
```
├── data/                   # Dataset files
├── model/                  # Trained ML models
├── notebook/              # Jupyter notebooks for analysis
│   ├── Cleaning.ipynb     # Data cleaning and preprocessing
│   └── ML_model.ipynb     # Machine learning model development
└── power bi/              # Power BI dashboard and visualizations
```

## Technologies Used
- Python
- Jupyter Notebook
- Machine Learning (scikit-learn)
- Power BI

## Setup
1. Clone the repository
2. Install required dependencies: `pip install -r requirements.txt`
3. Run the notebooks in the `notebook/` directory

## Results
View the Power BI dashboard visualizations in the `power bi/` folder for insights.
